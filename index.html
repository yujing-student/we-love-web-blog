sprint 12 bramus

Cross-document View Transitions voor MPA's Inleiding Cross-document view transitions zijn een nieuwe functie in CSS die het mogelijk maakt om naadloze overgangen te creëren tussen verschillende pagina's in een multi-page applicatie (MPA). Dit kan de gebruikerservaring (UX) aanzienlijk verbeteren door de overgang tussen pagina's soepeler en minder abrupt te maken.

Cross-document view transitions werken door snapshots te maken van de oude en nieuwe pagina's. Deze snapshots worden vervolgens gebruikt om een animatie te creëren die de overgang tussen de twee pagina's weergeeft. De animatie kan worden aangepast met behulp van CSS om het gewenste effect te creëren.

stappen view transition: stap 1 identify and name the view transition stap 2 geef ieder elemen een view transition name stap 3: trigger view transiotn in javascript stap 4 customige the animations using css
